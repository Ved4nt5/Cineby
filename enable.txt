Data Source Integration
Default API source:
text
https://cineby.at

The app should scrape/use Cineby's publicly available content endpoints and dynamically display movies and shows.
Dynamic Source URL System
Since Cineby's domain changes frequently, implement:
Source Manager
• Settings → Source Configuration.
• Allow user to change base URL manually.
• Example: 
◦ https://cineby.at
◦ https://cineby.xyz
◦ https://cineby.something
